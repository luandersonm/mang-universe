# 📚 Mangá Universe

## 📖 Descrição
O **Mangá Universe** é um catálogo online de mangás, criado para reunir os melhores títulos, desde lançamentos recentes até relíquias clássicas, tudo apresentado de forma gratuita e acessível. O projeto é focado em proporcionar uma experiência visual agradável e organizada para fãs de mangá.

## ✨ Funcionalidades
- 📔 Catálogo visual de mangás em destaque
- 🗂️ Seções separadas para diferentes gêneros ou destaques
- 🔎 Barra de pesquisa para facilitar a navegação
- 📱 Layout responsivo e moderno
- ℹ️ Créditos e informações no rodapé

## 🛠️ Tecnologias Utilizadas
- 🟧 HTML5
- 🟦 CSS3 (reset e estilos customizados)
- 🖼️ Font Awesome para ícones
- ⚡ Estrutura pronta para integração de JavaScript

## 🚀 Como Executar
1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Navegue pelo catálogo e explore os mangás disponíveis.
